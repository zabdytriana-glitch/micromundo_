1.1 — ¿Qué es la Micro:bit?
Teoría ≈ 20 min expand_more
lightbulb ¿Qué es la Micro:bit?

La Micro:bit V2 es una tarjeta de desarrollo creada por la BBC (Londres) diseñada para educación tecnológica. Tiene el tamaño de una tarjeta de crédito y contiene un microprocesador capaz de ejecutar programas, realizar cálculos y controlar dispositivos externos.

1.2 — Circuitos eléctricos básicos
Teoría ≈ 25 min expand_more
bolt Ley de Ohm y circuitos

Para conectar componentes a la Micro:bit necesitas entender tres conceptos clave: Voltaje (V) — la presión que empuja los electrones; Corriente (A) — el flujo de electrones; Resistencia (Ω) — frena el paso de la corriente.

V = I × R Ley de Ohm — V: voltios · I: amperios · R: ohmios · La Micro:bit trabaja con 3 Voltios
warning Todo circuito requiere dos puntos de conexión: un pin de datos (P0/P1/P2) y el pin GND para completar el circuito de retorno.
assignment Actividad: Calcular resistencia para LED
  • Un LED rojo necesita 1.8V y 20mA. La Micro:bit entrega 3V. Aplica V=IR para calcular la resistencia necesaria.
  • Solución: R = (3V − 1.8V) / 0.02A = 60 Ω. Valor comercial más cercano: 68 Ω.
  • En MakeCode: escribe el bloque escritura analógica pin P0 a 1023. Observa que el LED enciende al máximo.
  • Cambia el valor a 512. ¿El LED cambia de intensidad? Discute por qué con tu grupo (P-P-I).

2.1 — MakeCode: Programación por Bloques
Teoría + Práctica expand_more
extension Bloques fundamentales de MakeCode

Al iniciar: Se ejecuta UNA sola vez al encender la tarjeta. Ideal para inicializar variables.

Para siempre: Se ejecuta en bucle infinito. Ideal para leer sensores continuamente.

Al presionarse el botón A/B: Evento — solo se ejecuta cuando se presiona el botón.

Si agitado: Detecta movimiento con el acelerómetro de la Micro:bit.

Reto 2.1 — Sensores y temperatura 
// Al oprimir B → mostrar temperatura
input.onButtonPressed(Button.B, function() {
    basic.showNumber(input.temperature())
})

// Al oprimir A → cara feliz/triste según temperatura
input.onButtonPressed(Button.A, function() {
    if (input.temperature() > 15) {
        basic.showIcon(IconNames.Happy)
    } else if (input.temperature() > 35) {
        basic.showIcon(IconNames.Surprised)  // mucho calor
    } else {
        basic.showIcon(IconNames.Sad)
    }
})

2.2 — Variables y Contadores
Práctica expand_more
inventory_2 ¿Qué es una variable?

Una variable es un espacio con nombre en la memoria del dispositivo para guardar información y usarla después. Diferencia clave: nivel de sonido (lectura del sensor en ese instante) vs Nivel_sonido (variable que guarda ese dato).

Variables y contador de presiones 
let Nivel_sonido = 0
let N_veces = 0

basic.forever(function() {
    Nivel_sonido = input.soundLevel()   // guardar en variable
})
input.onButtonPressed(Button.A, function() {
    basic.showNumber(Nivel_sonido)      // usar variable
    N_veces += 1                         // incrementar contador
})
input.onButtonPressed(Button.B, function() {
    basic.showNumber(N_veces)            // cuántas veces presioné A
})
Promedio = SUMA ÷ N_lecturas Si tomaste 10 lecturas: Promedio = SUMA ÷ 10 — la media aritmética de los datos

2.3 — Bucles y Condicionales
Práctica expand_more
Algoritmo MÁXIMO en 10 lecturas 
let MAXIMO = 0
for (let i = 0; i < 10; i++) {
    let temp = input.temperature()
    if (temp > MAXIMO) {
        MAXIMO = temp        // actualizar máximo
    }
    basic.pause(1000)      // esperar 1 segundo
}

eco Para encontrar el MÍNIMO: inicializa la variable en 999 (muy alto) y cambia la condición a temp < MINIMO. ¡Pruébalo con los datos del dado: 2,5,3,1,4,5,3,6,6,1!

3.2 — El Efecto Fotoeléctrico y los Sensores de Luz
Física cuántica expand_more
science ¿Qué es el efecto fotoeléctrico?

Es el fenómeno en que un material emite electrones cuando recibe luz. Einstein (1905) lo explicó con su modelo cuántico — le valió el Premio Nobel de Física.

Modelo clásico (incorrecto): La luz es una onda — se pensaba que más brillo siempre arrancaría electrones.

Modelo cuántico (correcto): La luz está formada por partículas llamadas fotones. Si el fotón tiene suficiente energía (frecuencia alta), puede liberar un electrón.

Analogía: Los fotones son como canicas — si cada canica tiene suficiente energía (frecuencia alta), puede sacar al electrón. Si no, por más canicas que lances, no lo logras.

E = h · f E: energía del fotón · h: constante de Planck (6.626×10⁻³⁴ J·s) · f: frecuencia de la luz
lightbulb Aplicaciones en electrónica: Paneles solares ☀️ · Fotoresistencia (LDR) 🔦 · Cámaras digitales 📷 · Sensores de seguridad 🔐
1
¿Qué determina si un fotón puede liberar un electrón de un material?
A
La intensidad (brillo) de la luz
B
La frecuencia (color/energía) de la luz
C
La velocidad de la luz
D
El tamaño del material
✅ ¡Correcto! E = h·f — la frecuencia determina la energía de cada fotón. ¡Einstein Premio Nobel 1921!
❌ Es la frecuencia, no la intensidad. Un láser rojo brillante no puede arrancar electrones si su frecuencia es insuficiente.

4.1 — Práctica: Control de LEDs
Práctica expand_more
hardware Materiales físicos: Micro:bit + Placa de expansión (tipo L u horizontal) + 2 LEDs + Cables conectores.
assignment Práctica 1 — Parpadeo alternado de LEDs
  • Fase 1 (3s): LED1 y LED2 encienden alternados a alta velocidad (100ms de intervalo).
  • Fase 2 (3s): Ambos LEDs parpadean al mismo tiempo (500ms ON / 500ms OFF).
  • Fase 3 (5s): Los dos LEDs se apagan. Espera silenciosa.
  • El ciclo se repite indefinidamente con el bloque Para siempre.
Práctica 1 — Secuencia LEDs (3 fases) 
basic.forever(function() {
  // Fase 1: alternado rápido 3s (30 ciclos × 100ms)
  for (let i = 0; i < 30; i++) {
    pins.digitalWritePin(DigitalPin.P0, 1)
    pins.digitalWritePin(DigitalPin.P1, 0)
    basic.pause(100)
    pins.digitalWritePin(DigitalPin.P0, 0)
    pins.digitalWritePin(DigitalPin.P1, 1)
    basic.pause(100)
  }
  // Fase 2: juntos 3s
  for (let j = 0; j < 3; j++) {
    pins.digitalWritePin(DigitalPin.P0, 1)
    pins.digitalWritePin(DigitalPin.P1, 1)
    basic.pause(500)
    pins.digitalWritePin(DigitalPin.P0, 0)
    pins.digitalWritePin(DigitalPin.P1, 0)
    basic.pause(500)
  }
  basic.pause(5000) // Fase 3: apagado 5s
})

4.3 — PWM y Servomotores
Electrónica avanzada expand_more
waves PWM — Modulación por Ancho de Pulso

PWM es la técnica para simular diferentes niveles de energía usando señales digitales ON/OFF. El Duty Cycle (%) es el porcentaje de tiempo que la señal está encendida.

Los servomotores se controlan con pulsos de período fijo (20ms): pulso corto (~1ms) → 0°; medio (~1.5ms) → 90°; largo (~2ms) → 180°.

Duty Cycle (%) = (Tiempo ON ÷ Período total) × 100 50% duty cycle = señal encendida la mitad del tiempo

5.1 — Especificaciones del sistema (Reto)
Proyecto integrador expand_more
  • Toma de datos: Temperatura, humedad (pin externo escalado) y nivel de luz — cada hora (simulada como 1 minuto = 60 000 ms).
  • Botón B: Inicia la toma continua de datos en bucle.
  • Botón A: Muestra los datos máximos, mínimos y el promedio de todas las variables acumuladas hasta el momento.
  • Alarma: Si la humedad baja del umbral definido → activa alarma sonora (music.playTone). El umbral es una variable configurable por programa (UMBRAL_HUM).
Prototipo Invernadero — código base 
let MAXIMO_T = 0, MINIMO_T = 999
let SUMA_T = 0, N = 0
let UMBRAL_HUM = 30   // % de humedad mínima aceptable
let midiendo = false

input.onButtonPressed(Button.B, function() {
    midiendo = true   // iniciar toma de datos
})

basic.forever(function() {
    if (midiendo) {
        let T = input.temperature()
        let H = pins.map(pins.analogReadPin(AnalogPin.P0), 0, 1023, 0, 100)
        N += 1;  SUMA_T += T
        if (T > MAXIMO_T) MAXIMO_T = T
        if (T < MINIMO_T) MINIMO_T = T
        if (H < UMBRAL_HUM) music.playTone(440, 500)  // alarma!
        basic.pause(60000)  // esperar 1 minuto
    }
})

input.onButtonPressed(Button.A, function() {
    basic.showString("MAX:"); basic.showNumber(MAXIMO_T)
    basic.showString("MIN:"); basic.showNumber(MINIMO_T)
    basic.showString("AVG:"); basic.showNumber(SUMA_T / N)
})

🖊️ Cómo usar esta plantilla en Google Sites

Este archivo HTML replica visualmente un Google Site completo. Para convertirlo en un Google Site editable sigue estos pasos:

  • 1
    Ve a sites.google.com e inicia sesión con tu cuenta de Google.
  • 2
    Haz clic en "+" para crear un nuevo sitio. Elige el tema de color Azul oceánico para que coincida con este diseño.
  • 3
    Crea una página por cada sección: Inicio · Módulo 1 · Módulo 2 · Módulo 3 · Módulo 4 · Módulo 5 · Recursos · Evaluación.
  • 4
    En cada página, usa los bloques de Google Sites: Texto expandible para lecciones · Tabla para datos · Código (embed HTML) para bloques de código · Imagen para diagramas.
  • 5
    Para los quizzes, incrusta Google Forms usando el bloque "Insertar → Formularios". Crea un formulario por módulo con las preguntas de opción múltiple.
  • 6
    Para el seguimiento de progreso, integra Google Sheets con las respuestas del formulario usando el bloque "Insertar → Hoja de cálculo".
  • 7
    Publica el sitio con Publicar → Configurar acceso como "Cualquier persona en tu organización" o "Cualquier persona con el enlace".
